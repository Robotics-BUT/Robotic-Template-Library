Detailed Description

template<typename Element>
class rtl::Polygon3D< Element >

Three dimensional polygon class.

For now, it aggregates points in std::vector, stores data of the plane in which they lie and allow transformation by Transformation3D. Extensions such as intersection with another polygon etc. are planned in the future.

Adds vertices from an iterable object using iterators.

Traverses iterators pointing to VectorType vertices by incrementation of beg until the end is reached and adds directly to the internal buffer. No checks, whether the vertices lie in the polygon plane are performed, so it is possible to make an invalid polygon that way. Use for increased speed when vertices in the range beg to end are guaranteed to be in the polygon plane.
