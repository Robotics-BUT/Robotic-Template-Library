Detailed Description

LaTeX export of high quality vector graphics using the Tikz package - 2D drawing.

This class is used to aggregate graphic primitives to be rendered into a PDF format. The rendering order corresponds to the order in which the primitives are added, with an exception of axes and grids, which are always in the bottom. The graphics is gradually built using add* functions and exported to TEX file when it is finished.
