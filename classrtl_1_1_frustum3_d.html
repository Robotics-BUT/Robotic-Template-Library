Detailed Description

template<typename Element>
class rtl::Frustum3D< Element >

3D frustum representation.

Fast implementation of a frustum object for storing detections from image data. The class does not perform any chcecks of data during construction, so it is possible to construct an invalid frustum that way. On the other hand, the implementation is very fast and requires minimum computational overhead.
