Detailed Description

template<typename Element, typename Compute>
class rtl::ApproximationTlsPlane3D< Element, Compute >

Planar approximation of a set of points in 3D.

Instances of ApproximationTlsPlane3D work as functor taking PrecSumsType argument and computing linear approximation based on it. Basic operations such as trim() to line segments or project() point are present as well.

Trims the approximation plane to a polygon using iterator-defined set of points.

Function with this signature is common to all kinds of approximation and is suitable for generic code. All points given by interval beg to end are projected onto *this and for an outline of the resulting polygon.
