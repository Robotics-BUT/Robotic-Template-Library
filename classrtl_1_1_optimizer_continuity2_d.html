Detailed Description

template<class SumArray, class Approximation>
class rtl::OptimizerContinuity2D< SumArray, Approximation >

Checks intersections and fixes 2D line approximations, if continuous output polyline is required.

If a complex shape of an ordered point cloud is vectorized, approximations near inflexion points of the cloud might not intersect close to the cloud (in principle they might not intersect at all). If a continuous polyline is required as the vectorization output, these artifacts can be fixed by addition of an extra approximation bridging the inflexion point. Requires array of precomputed sums to work.

Sets maximal permitted distance of intersection of the approximation lines from their neighbouring end points.

delta should be circa 3-10 times larger that standard deviation used in preceding vectorization to avoid unnecessary splits of approximations and maintain functionality of the optimization procedure.
