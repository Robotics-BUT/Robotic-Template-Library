Detailed Description

template<typename Obj, typename Tf>
struct rtl::is_transformable< Obj, Tf >

Tests if an object of type Obj can be transformed by a transformation of type Tf.

For transformation of an object by rtl::GeneralTf, the value is true, if at least one of the alternative transformations gives valid result (others might be invalid).

Template Parameters
Objtype of the object to be transformed.
Tftype of the transformation applied.
