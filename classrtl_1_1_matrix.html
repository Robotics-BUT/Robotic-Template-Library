Detailed Description

template<int rows, int cols, typename Element>
class rtl::Matrix< rows, cols, Element >

Template for general Matrices of both static and dynamic nature.

Static matrices are constructed using positive integer in place of rows and cols, while dynamic use Eigen::Dynamic at these places.

Template Parameters
rowsnumber of matrix rows specified at compile time, or Eigen::Dynamic.
colsnumber of matrix columns specified at compile time, or Eigen::Dynamic.
Elementbase type of matrix elements.
