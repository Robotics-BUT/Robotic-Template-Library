Detailed Description

template<class SumArray, class Approximation>
class rtl::ExtractorChainFast< SumArray, Approximation >

Extracts geometrical primitives from an ordered point cloud.

Approximation fitting works on the binary search principle - to fit an approximation to the longest section of N point chain, roughly log N computations is required.

Template Parameters
SumArraytype of the array of precomputed sums used.
Approximationtype of approximation to be fitted to the data.

Functor call for processing of an array of precomputed sums.

Output parameters are always cleared before the extraction process. Order of extracted primitives corresponds to their order in the precomputed sums (and therefore in the point cloud). Corresponding approximations and indices have the same index in output vectors.
