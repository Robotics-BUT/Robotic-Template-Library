Detailed Description

LaTeX document for high-quality human-readable output.

LaTeX document allows to aggregate other LaTeX outputs into a single document and provides better unified methods for dealing with compilation commands, temporary files etc. It is possible to automate creation of complex documents such as reports from experiments, or mass data processing with the top visual quality LaTeX can offer.

Finalize the document and destroy *this.

In the destructor the document is finalized and compiled. After that, the move_final_doc() and remove_tmp_dir() internals are called and *this is finaly destroyed.
