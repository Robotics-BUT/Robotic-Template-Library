Detailed Description

template<typename Compute, class SumType>
struct rtl::PrecArayBase< Compute, SumType >

Base class for precomputed arrays.

Common operations for all precomputed arrays are implemented here. Adds the first row of zeros to the array making the array one row longer than the number of points used to its precomputation. This feature makes computation of precomputed sums as simple as subtraction of two rows of the array in all cases.
